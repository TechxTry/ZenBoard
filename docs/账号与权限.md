## 账号与权限（多用户版）

### 账号类型说明
- **system_users**：本系统登录账号（管理员创建、可禁用、可重置密码）。
- **local_users**：从禅道同步到本地 PostgreSQL 的人员镜像（用于数据统计与筛选），不是登录账号。

### 角色（RBAC）
- **super_admin**：系统最高权限（用户/绑定/配置/同步等管理操作）。
- **admin**：管理权限（同上，后续可细分）。
- **user**：普通用户（仅能访问授权范围内的数据）。

### 数据范围（Data Scope）
- **SELF**：仅本人数据（以“绑定的禅道账号”作为口径）。
- **GROUP**：仅默认小组数据（`default_group_id` 对应的小组成员）。
- **ALL**：全量数据（通常仅管理员）。

### 禅道账号绑定
系统用户与禅道账号采用 **1:1** 绑定关系（禁止复用）。绑定后用于：
- 默认个人口径（SELF）筛选
- 部分页面与查询的权限裁剪（后端强制）

### 越权防护原则
- **前端隐藏菜单只做体验优化**，真正的权限控制以**后端**为准。
- 后端对 `workbench/*`、`analytics/*` 等接口会根据 `data_scope` 强制裁剪参数与可见数据范围。
